⚠️ Limitations du pipeline actuel
🔴 Ce qui manque dans ce pipeline à 2 stages
🔍
Qualité du code absente
Aucun SonarQube, ESLint, Checkstyle ni Quality Gate. La dette technique s'accumule sans détection.
🔐
Aucun scan de sécurité CI
Trivy est appliqué manuellement par l'équipe Ops — trop tard pour bloquer une image vulnérable.
🐳
Pas de packaging automatisé
Les images Docker sont construites et publiées manuellement sur Docker Hub.
🚀
Aucune livraison continue
Pas de déploiement automatique. Tout est déclenché manuellement par l'équipe Ops.
🏷️
Versionnage manuel
Pas de Semantic Release. Les numéros de version sont gérés à la main (v1.2.3 en dur).
🔑
Détection de secrets absente
Aucun scan Gitleaks. Des credentials pourraient être exposés dans le code source.
