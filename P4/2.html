⚙️ Configuration Base de Données & Frontend

Vérification de la connexion à la BD et lancement du Frontend

1️⃣ Fichier de connexion à la Base de Données

Le fichier de connexion à la base de données est correctement configuré. Les paramètres d’accès (URL, utilisateur, mot de passe) sont bien définis et permettent au backend de communiquer avec la BD.

✔ Connexion à la base de données configurée

2️⃣ Démarrage du Frontend

Le frontend démarre correctement avec l’une des commandes suivantes :

npm run start
ou
ng serve

Après l’exécution de la commande, l’application frontend est accessible depuis le navigateur.

✔ Le frontend démarre bien
