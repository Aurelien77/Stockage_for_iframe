Connexion du projet à PostgreSQL

Étape 1 : Installer le driver PostgreSQL

Selon votre langage / framework, installez le driver PostgreSQL :
Node.js : npm install pg
C# (.NET) : Install-Package Npgsql

Étape 2 : Configurer la connexion

Créez un fichier de configuration ou modifiez votre fichier existant pour connecter votre projet :

// Exemple Node.js const { Client } = require('pg'); const client = new Client({ user: 'postgres', host: 'localhost', database: 'votre_base', password: 'votre_mdp', port: 5432, }); client.connect();

Étape 3 : Vérifier la connexion

Testez la connexion dans votre projet :

// Node.js client.query('SELECT NOW()', (err, res) => { if(err) { console.error('Erreur de connexion', err); } else { console.log('Connexion OK:', res.rows); } client.end(); });

Étape 4 : Conseils supplémentaires

  • Assurez-vous que le service PostgreSQL est démarré
  • Vérifiez le mot de passe et le port
  • Pour les frameworks web, utilisez des variables d’environnement pour stocker les credentials
  • En cas de firewall Windows, autorisez le port 5432
