Installation et démarrage de PostgreSQL sur Windows 11

Étape 1 : Télécharger PostgreSQL

Rendez-vous sur le site officiel : https://www.postgresql.org/download/windows/ et téléchargez l'installateur version 18 pour Windows 11.

Étape 2 : Installation

Lancez l'installateur téléchargé et suivez les instructions :

  • Choisir le dossier d’installation
  • Définir un mot de passe pour l’utilisateur postgres
  • Sélectionner les composants souhaités

Étape 3 : Vérification du service

Pour vérifier que PostgreSQL est bien démarré sur Windows 11 :

  1. Ouvrir le Gestionnaire des services (Win + R → services.msc)
  2. Chercher postgresql-x64-18 ou similaire
  3. Vérifier que le statut est Démarré

Étape 4 : Connexion à la base

Vous pouvez utiliser pgAdmin ou la ligne de commande : 

psql -U postgres -h localhost -p 5432
Entrez le mot de passe défini lors de l’installation pour accéder à votre base.
