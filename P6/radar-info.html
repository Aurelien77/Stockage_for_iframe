Radar DevOps IA

Cette plateforme a été développée dans le cadre du Projet 6 OpenClassrooms. Elle permet à un expert DevOps de réaliser une veille technologique et une étude comparative des technologies utilisées dans une application.

Accéder au Radar
Objectif du projet

Le radar technologique aide les équipes techniques à analyser et comparer les technologies utilisées dans leurs projets afin de faciliter les décisions d’architecture logicielle.

Analyse des technologies

Le radar identifie les technologies utilisées et analyse leur maturité et leur adoption.

Comparaison des outils

Les technologies peuvent être comparées afin d'identifier les solutions les plus pertinentes.

Veille technologique

Le radar permet de suivre l’évolution des outils DevOps et technologies modernes.

Analyse par Intelligence Artificielle

Une intelligence artificielle analyse les technologies et génère des recommandations pour améliorer l’architecture d’un projet.

Scan des technologies

L’IA détecte automatiquement les technologies présentes dans l’application.

Évaluation de fiabilité

Chaque technologie reçoit un score basé sur sa maturité et son adoption.

Suggestions

L’IA propose des alternatives ou nouvelles technologies à tester.

Fonctionnement du radar
01

Analyse du projet

Le système identifie les technologies utilisées.

02

Analyse IA

03

Comparaison

Les technologies sont positionnées sur le radar.

04

Décision technique

Les experts DevOps prennent les décisions.

Collaboration et documentation

Documents de formation

Des ressources pédagogiques peuvent être ajoutées pour former les équipes.

Commentaires d’équipe

Les développeurs peuvent partager leurs retours et expériences.

Décisions techniques

Les discussions permettent d’améliorer les choix technologiques.
